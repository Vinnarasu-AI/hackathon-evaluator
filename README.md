# Hackathon Project Evaluator

AI-powered tool to automatically evaluate hackathon projects based on 13 criteria.

## Features
- Clone any public GitHub repository
- Analyze code using AI (Llama3 for evaluation, nomic-embed-text for embeddings)
- Generate scores for 13 evaluation criteria
- Export results as JSON or PDF
